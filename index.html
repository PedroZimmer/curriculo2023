ExcelToOFX - ZPD Processamento de Dados

Desenvolvi um software em Python que automatiza a conversão de um formato de arquivo XLSX para OFX. Esta solução tem se mostrado altamente eficaz na redução do tempo dedicado à tarefa de digitação manual de extratos bancários e boletos. Anteriormente, o processo demandava um período de 2-3 semanas para ser concluído manualmente, enquanto que, com o uso deste software, o tempo necessário foi reduzido significativamente, atualmente levando apenas 10 minutos para executar a mesma tarefa.

Somos Agape - UNIVILLE

Em parceria com um colega, estou atualmente envolvido no desenvolvimento de um site institucional para a organização beneficente "Somos Ágape", de Joinville. O objetivo primordial é criar uma presença online atraente e informativa, que seja capaz de cativar potenciais patrocinadores e interessados em conhecer mais sobre o instituto. Além disso, planejamos implementar uma área restrita dentro do site para permitir que os administradores tenham controle sobre as finanças da instituição, juntamente com a incorporação de diversos outros requisitos que serão identificados durante o processo de desenvolvimento. As tecnologias utilizadas até o momento incluem HTML, CSS, JavaScript e Bootstrap para a construção da interface. Em fases posteriores do projeto, estamos contemplando a integração de tecnologias Java para o desenvolvimento do backend, bem como a implementação de um banco de dados para suportar as funcionalidades necessárias.
